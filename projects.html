How to toggle a boolean using JavaScript?

The boolean is toggled whenever the button is pressed.

window.onload = function checkClick() { const darkThemeSelected = localStorage.getItem('click') == true && localStorage.getItem('darkSwitch') === 'dark'; darkSwitch.clicked = darkThemeSelected; darkThemeSelected ? document.body.setAttribute('data-theme', 'dark') : document.body.removeAttribute('data-theme'); }
